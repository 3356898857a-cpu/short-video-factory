# 短视频内容工厂 🎬

> 专为抖音矩阵号设计的每日内容策划工具。一键 AI 生成选题、脚本、BGM，追踪对标账号，管理发布流程。

## 功能概览

- **📋 今日工作台** — 一键 AI 生成今日 5 个账号的视频策划（选题 + 脚本 + BGM + 对标参考 + 拍摄建议）
- **👤 账号管理** — 管理抖音账号矩阵，设置赛道、人设、对标账号
- **🎯 对标账号库** — 追踪竞品爆款视频，AI 自动参考
- **📅 内容日历** — 按日期回顾历史内容，查看播放数据
- **⚙️ 设置** — API Key 配置、数据导入导出备份

## 使用方法

### 1. 打开网页

**方法一：直接双击打开（本地使用）**

下载 `index.html`，双击用浏览器打开即可。

**方法二：部署到线上（手机电脑都能用）**

部署到 GitHub Pages（免费）：

```bash
# 1. 在 github.com 创建一个新仓库（如 short-video-factory）
# 2. 把 index.html 上传到仓库
# 3. Settings → Pages → 选 main 分支 → Save
# 4. 访问 https://你的用户名.github.io/short-video-factory/
```

### 2. 初始设置

1. 打开网页 → 点击底部「设置」Tab
2. 填入你的 DeepSeek API Key（去 platform.deepseek.com 申请）
3. 设置默认赛道和 BGM 风格偏好

### 3. 每日工作流程

```
每天早上:
  1. 打开网页 → 点「✨ AI 生成今日内容」
  2. 等待 10-20 秒 → 5 条视频策划卡片出现
  3. 查看每条：选题、脚本、BGM、对标参考、拍摄建议

拍摄时:
  4. 点「复制全文」→ 粘贴到备忘录 → 照着念
  5. 拍摄完成后 → 点「已拍摄」更新状态

剪辑时:
  6. 选好 BGM → 剪映自动字幕 → 点「剪辑中」

发布后:
  7. 点「已发布」→ 填入播放量和点赞数
  8. 去「日历」Tab 查看数据积累
```

### 4. 对标账号管理

1. 去「对标」Tab → 「添加对标账号」
2. 为每个对标账号添加爆款视频（链接 + 选题 + 可借鉴点）
3. AI 生成时会自动参考这些数据找差异化角度

### 5. 数据备份

- 设置 Tab → 「导出数据」→ 下载 JSON 文件
- 换电脑时 → 「导入数据」→ 选择 JSON 文件恢复
- 建议每周导出一次备份

## 技术说明

- **数据存储**：所有数据保存在浏览器本地（localStorage），不上传任何服务器
- **API 调用**：仅将账号信息和对标数据发送给 DeepSeek API 用于生成内容
- **费用**：每次生成约消耗 500-4000 tokens，约 0.0065 元/次，一个月不到 2 毛钱
- **隐私**：API Key 存储在浏览器本地，仅发送给 DeepSeek 官方 API

## 常见问题

**Q: 点生成按钮没反应？**
A: 先去「设置」Tab 检查 API Key 是否已填入且有效。

**Q: 生成的脚本不合适？**
A: 可以点状态按钮标记为「待拍摄」然后再次生成，新内容会替换当天的。也可以在账号管理中调整人设定位让 AI 更精准。

**Q: 数据会不会丢？**
A: 数据存在浏览器 localStorage 中，清除浏览器缓存会丢失。建议定期去设置导出 JSON 备份。

**Q: 能用手机打开吗？**
A: 可以。部署到 GitHub Pages 后手机浏览器也能访问，底部 Tab 栏适配了移动端。
